## 補足 (返信用 / 質問が来たら)

**なぜ Slack?**
消防団がすでに連絡用に Slack を使っている。新しいアプリを入れさせない・Web 画面を見せない、で「出動の瞬間」に届くチャネルがそこしかない。

**スコープ最小化**
- Cloudflare Workers + KV + Gemini API + Slack Events API のみ
- 「1画面のチェックリスト」を「1つのスレッド返信」に圧縮
- 出動分類は keyword fast path、不確実なら Gemini に投げる
- 学習は Gemini multimodal が画像から教訓1文+キーワードを JSON で返す

**安全性**
個人の訓練リコール補助。正式な指揮系統・SOP・公式マニュアルの代替ではない (Bot メッセージにも毎回フッターで明示)。

**ロケーション秘匿**
実在の消防団保護のため、地名ではなく「気候 × 地形」プリセット (高地砂漠台地 / 沿岸温帯 / 山岳森林 / WUI) でしか動かない。

**次に伸ばすなら**
- 訓練ログの累積を週次サマリで返す (VPC の ★Gain Creator 強化)
- スラッシュコマンドで現場タイプ別 SOP を呼び出せるようにする
- 複数チーム対応 (`teamId` キーは入っているが UI 未整備)
